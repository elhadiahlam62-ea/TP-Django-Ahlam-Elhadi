# TP Django - Word Counter & Authentification

Ce projet a été réalisé dans le cadre du TP de Django à la **FST de Tanger**.

## Fonctionnalités
- **Compteur de mots :** Analyse un texte et retourne le nombre de mots.
- **Système d'Authentification :** Inscription, connexion et déconnexion des utilisateurs.
- **Base de données :** Utilisation de SQLite pour stocker les informations.

## Installation
1. Cloner le dépôt.
2. Créer un environnement virtuel : `python -m venv monenv`.
3. Installer Django : `pip install django`.
4. Lancer les migrations : `python manage.py migrate`.
5. Lancer le serveur : `python manage.py runserver`.
